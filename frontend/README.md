# Frontend

Vite + React frontend for the dashboard project.

## Environment Variable

Create `frontend/.env`:

```env
VITE_API_URL=http://localhost:5000/api
```

For Vercel, set `VITE_API_URL` to your deployed Railway backend URL with `/api` appended.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```
